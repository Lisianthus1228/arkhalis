# Keyboard Shortcuts
- These can be held to buy many "buildings" very quickly.

G: Buys guide.  
E: Buys enchanted sword.  
N: Buys night's edge.  
B: Buys broken hero sword.  
H: Buys limit upgrade to broken hero swords.
